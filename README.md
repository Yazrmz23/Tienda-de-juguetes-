Estructura básica de una aplicación web.e una tienda de juguetes.
El proyecto explora cómo utilizar tecnología descentralizada para registrar productos, gestionar ventas y realizar transacciones seguras dentro de una plataforma digital.
El objetivo principal es crear una aplicación sencilla que represente el funcionamiento de una tienda en línea. A través de este proyecto se busca comprender cómo se pueden integrar herramientas de desarrollo web con conceptos de blockchain para gestionar productos y transacciones digitales.
